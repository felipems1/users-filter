# Users Filter

## Tecnologias

- Angular
- TypeScript
- Tailwind CSS
- Angular Material
- Date-Fns

## Instalação

1. Clone este repositório:

   ```bash
   git clone https://github.com/felipems1/users-filter.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd users-filter
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
## Rodando projeto

1. Execute o seguinte comando para inicializar a aplicação:

   ```bash
    npm run start
   ```
